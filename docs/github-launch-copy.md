# GitHub Launch Copy

This file captures the recommended public-facing copy for the first release.

## Recommended Repository Name

`governed-skill-tree`

Why this one:

- clear on first read
- close to the core idea
- not locked to Unity
- not locked to one model vendor

## Recommended GitHub Short Description

`A governed skill system for long-horizon human-AI collaboration, with explicit routing, phase control, verification, and gradual evolution.`

## Recommended README Title

`Governed Skill Tree`

## Recommended First Paragraph

`Governed Skill Tree is an organ-based skill system for human-AI collaboration. It separates routing, phase control, tool choice, risk interception, verification, memory, and communication into explicit governed layers so a skill tree can grow without turning into prompt soup.`

# Release Scope

This repository is a first public batch, not a dump of every internal artifact.

## Included On Purpose

- the governed core
- the metadata template
- routing, phase control, and tool-surface separation
- assumption and dependency governance
- one mature domain showcase

## Held Back On Purpose

- the full internal skill inventory
- every compatibility bridge
- all workflow-specific tooling surfaces
- high-noise or low-signal internal experiments

## Why The Scope Is Narrow

The goal of the first release is clarity.

It should communicate:

- what the system is
- why it is different
- how it stays maintainable
- how one vertical pack plugs into the generic core

It should not require the public reader to absorb every internal workflow before understanding the design.

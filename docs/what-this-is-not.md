# What This Is Not

This project can be misunderstood if it is read too quickly.

Here is what it is not trying to be.

## Not A Prompt Dump

The point is not to collect as many prompts as possible.

The point is to separate ownership:

- routing
- phase control
- tool choice
- risk interception
- execution
- verification
- communication

## Not A Universal Replacement For Agent Frameworks

This project is about skill governance.

It can complement larger agent frameworks, but it is not trying to replace every orchestration runtime, memory system, or evaluation stack.

## Not A Unity-Only Repository

The first showcase pack happens to be Unity-focused because it is mature and evidence-rich.

The core ideas are broader than Unity:

- authority boundaries
- intake pipeline
- metadata contracts
- governed growth

## Not "Just Use Better Naming"

Naming helps, but names alone do not solve:

- overlapping authority
- phase confusion
- unsafe escalation
- unclear migration

Governance rules are part of the design, not an afterthought.

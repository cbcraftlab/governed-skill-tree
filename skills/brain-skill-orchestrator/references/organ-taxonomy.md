# Organ Taxonomy

Use this map to decide which category should lead the task.

## Brain

Use for:

- broad goals
- ambiguous requests
- mixed requests needing routing

## Controller

Use for:

- stage transitions
- deciding whether to explore, implement, validate, or hand off next

## Eyes

Use for:

- captures
- runtime behavior
- visual or behavioral comparison

## Ears

Use for:

- subjective product feedback
- fuzzy user language that needs interpretation

## Memory

Use for:

- recall
- save
- handoff
- compression

## Hands

Use for:

- code edits
- shell operations
- editor actions

## Legs

Use for:

- sustained rollout
- verification sequencing
- multi-step progress

## Balance

Use for:

- safety checks
- review
- regression risk
- scope control

## Collaboration

Use for:

- summaries
- handoff
- explanations
- delivery notes

# State Model

Use this note when the correct current phase is ambiguous.

## Explore

Signals:

- still collecting files, logs, captures, or reproduction steps
- still unsure which subsystem owns the issue
- still comparing hypotheses

## Decide

Signals:

- the problem is understood enough to pick a bounded approach
- affected files or boundaries are mostly known
- risk level can be named

## Implement

Signals:

- a concrete change is being applied
- tools are operating on files, scenes, or environment state

## Validate

Signals:

- compiling, testing, smoking, comparing outputs, or checking regressions

## Handoff

Signals:

- preparing a continuation note
- compressing context for another window or another account
- pausing work intentionally

## Archive

Signals:

- requested goal is complete
- residual risks are called out
- no immediate next work is expected

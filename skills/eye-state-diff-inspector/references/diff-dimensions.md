# Diff Dimensions

Use these dimensions to organize differences.

## Data

- values
- ids
- ordering
- missing or extra records

## Configuration

- flags
- thresholds
- environment settings
- runtime options

## UI State

- active/inactive
- visible/hidden
- focus state
- selected mode

## Runtime Behavior

- timing
- sequence
- transitions
- side effects

## Layout or Visual

- positioning
- size
- line wrapping
- scroll position

## Noise Filter

Treat these as possible noise unless they explain behavior:

- timestamps
- unstable ordering with no visible effect
- known expected cosmetic drift
